# EduTech - UKM Gradien

💻 **Program Pembelajaran Web Development**
📚 HTML • CSS • JavaScript
🚀 Department Education & Training - UKM Gradien

---

## 📖 Tentang EduTech

EduTech merupakan program kerja dari Department Education & Training UKM Gradien yang berfokus pada pengembangan kemampuan anggota di bidang teknologi, khususnya Web Development.

Kegiatan ini dirancang sebagai wadah pembelajaran interaktif dan praktik langsung (*hands-on learning*) untuk membantu peserta memahami dasar pembuatan website menggunakan HTML, CSS, dan JavaScript.

---

## 🎯 Tujuan Kegiatan

* Meningkatkan kemampuan dasar web development anggota UKM Gradien
* Memberikan pengalaman belajar berbasis praktik
* Mengembangkan kreativitas dan problem solving
* Membantu peserta memahami pengembangan website sederhana
* Membangun lingkungan belajar teknologi yang kolaboratif

---

## 📚 Materi Pembelajaran

### 🌐 HTML

* Struktur dasar HTML
* Heading, paragraf, gambar, dan link
* Form dan tabel
* Layout dasar website

### 🎨 CSS

* Styling dasar
* Warna dan typography
* Flexbox dan layout
* Responsive design

### ⚡ JavaScript

* Variabel dan tipe data
* Function dan event
* DOM Manipulation
* Interaktivitas website
